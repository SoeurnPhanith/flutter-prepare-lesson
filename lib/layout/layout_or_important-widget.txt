            ney horizontal bos Row ng hx vea mean value(start, center,end

        2.  crossAxisAlignment : user for manage ler position ney vertical

3.Stack?
    =>.Stack គឺជា layout widget ដែលអនុញ្ញាតឱ្យ widget 
    ច្រើន ដាក់ត្រួតលើគ្នា (overlap) ដូចជា layer។
+Syntax
    Stack(
        children: [
            Widget1,
            Widget2,
            Widget3,
         ],
    )

+propoties using in stack : [
    1.children : dak widget ler knea ban
    2.fit : manage size rbos non-positional widget
    3.alignment : define ទីតាំង default របស់ children
    4.clipBahavoir : កាត់ (clip) widget ដែលលើស Stack
]

+Widget using in stack : {
    1.  Positioned : use for set absulute position or specific position
     of child and having propoties : [
      +top : space on top
        +bottom : space on bottom
        +left : space on left
        +right : space on right
        +width : define width
        +height : define height
    ]
    2.  Align : use for set relative postion of widget and it having
        propoties : [
            +alignment : Alignment.bottomRight
            +alignment : Alignment.topLeft
            +alignment : Alignment.center
        ]
    3.  Opacity : use for make widget ស្រអាប់ / ថ្លា and having
    propoties : [
        +opacity : value (double)
    ]
    4.  ClipRRect/ ClipOval : use for កាត់ shape (rounded / circle)
    5.  Tranform : user for Rotate / Scale / Move widget
    6.  IgnorePointer widget ដែលប្រើសម្រាប់
    មិនអនុញ្ញាតឱ្យ child widget ទទួល event touch / click and having propoties [
        +ignore : true == can not click 
        +ignore : false == can click
    ]
}


4.Container?

✅ កំណត់ size widget
✅ បន្ថែម background color / border / radius

✅ បង្កើត UI card / box / panel

    +propoties use in Container : [
        1.width : set width
        2.height : set height
        3.child : use for put one more Widget on it
        4.padding : inner space of container
        5.margin : outer space of container
        6.alignment
        7.decoration : use for design or more implement to container like:{
            +color  
            +borderRaduis
            +border
            +boxShadow
            +gradient
            +Tranform
            +clipBahavoir
        }
    ]


5.Expanded?
    =>Expanded គឺជា widget ដែល បង្ខំ child ឲ្យ យក
     space ទាំងអស់ដែលនៅសល់ ក្នុង Row, Column, ឬ Flex.

+ប្រើ Expanded នៅពេល:
    -ចង់ឲ្យ widget ពង្រីកពេញ space
    -បង្ការ overflow error
    -បែងចែក space ស្មើៗគ្នា


6.Flexible?
    =>Flexible គឺជា widget ដែល អនុញ្ញាត child ឲ្យ យក space តែប៉ុណ្ណោះដែលវាត្រូវការ
     មិនបង្ខំឲ្យ fill space ទាំងអស់.


+Expanded vs Flexible (Key Difference ⭐)
.Feature	                Expanded	         Flexible
.Fill space  	            ✅ Always	        ❌ Not always
.Default fit	            tight	             loose
.Overflow safe       	    ✅	                ✅
.Control size	            ❌	                ✅
