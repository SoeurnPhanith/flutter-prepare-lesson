1.Row ? 
    => Row jea widget muy deal dak oy child bos vea tv jea jour dek 
(horizontal).
    +Syntax : 
    Row(
        children: [
            Widget1(),
            Widget2(),
            Widget3(),
        ],
    )

    +Importance propoties in Row : 
        1.  mainAxisAlignment : use for manage ler position of children
            ney horizontal bos Row ng hx vea mean value(start, center,end
            spaceBetween : ចែកចន្លោះគ្នាដោយប្រើ empty space,
            spaceAround : ចន្លោះត្រូវបានចែកគ្នាបានសមគ្នា,
            spaceEvendly: ចែកជាកម្រិតស្មើគ្នា)

        2.  crossAxisAlignment : user for manage ler position ney vertical
            bos Row ng hx mean value (start,center,end)

        3.mainAxisSize : កំណត់ទំហំ Row តើត្រូវ full width ឬ ត្រឹមតែប៉ុណ្ណោះ?
