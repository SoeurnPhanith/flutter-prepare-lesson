3. Command useful in flutter buildig!
    ->flutter doctor                 : use for check flutter installation 
    ->flutter create project-name    : use for create project
    ->flutter run                    : use for run flutter project
    ->flutter device                 :  Check all device name having on machine
    ->flutter emulators
      flutter emulators --launch <emulator_id> : use for open emulator
    ->flutter pub get                : use for get or install package again
    ->flutter pub upgrade            : use for updrade package
    ->flutter clean                  : use for clean project or clear cache
    ->flutter build apk              : use for deploy private app to user(android)
    ->flutter build ios              : use for deploy private app to user(IOS)
    ->flutter build appbundle        : build app to play store or app store
    ->flutter analyze                : Analyse code
    ->flutter --version              : check flutter version
    ->flutter format .               : use for format code to pretty code
